2. CHESS


A simple web-based chess game where you can play as White against a basic AI opponent. Built with HTML, CSS, and JavaScript, this project demonstrates a chessboard interface, move validation, and a basic AI using a simple Minimax-like evaluation.
🎮 Features
Interactive 8×8 chessboard with clickable squares.
Legal move highlighting for selected pieces.
Basic AI opponent that evaluates board positions and chooses moves.
Move history panel showing all moves in algebraic notation.
Responsive design that fits any screen size.
🖥️ Demo
<img width="1342" height="660" alt="Screenshot 2026-01-30 113202" src="https://github.com/user-attachments/assets/09f20b70-53b7-4436-b940-5e5ebeea61e2" />
🚀 How to Play
Clone or download this repository.
Open index.html in your browser.
Play as White by clicking on your pieces and then selecting a legal move.
The AI (Black) will automatically make a move after yours.
Continue until checkmate, stalemate, or resignation.
⚙️ How It Works
The board is represented as an 8×8 array, with uppercase letters for White and lowercase letters for Black.
Legal moves are generated for each piece based on standard chess rules (simplified, no castling, en passant, or promotion yet).
The AI evaluates each potential move using a basic material score and chooses the move with the highest evaluation.
Moves are logged in a panel using standard algebraic notation for easy tracking.
Core Components
HTML: Chessboard layout and move panel.
CSS: Styling for board squares, highlights, and overall layout.
JavaScript:
Game state management (board, turn, selected)
Rendering the board and highlighting legal moves
Handling user clicks and moves
Basic AI logic and evaluation
📝 Limitations & TODOs
AI is basic; only evaluates material, not position or strategy.
No special moves: castling, en passant, pawn promotion not implemented.
No check/checkmate detection yet.
Future improvements could include minimax with depth, alpha-beta pruning, and full chess rules.

🎮 Features
Two Game Modes:
Player vs Player
Player vs Computer (AI uses Minimax algorithm – unbeatable!)
Modern & responsive design for desktop and mobile
Visual highlighting of winning cells
Restart button to quickly start a new game
Dynamic status updates showing turns, wins, or draws
Friendly pink and dark theme with smooth animations
🕹 How to Play
Select a mode from the dropdown (Player vs Player or Player vs Computer).
Click on a cell to place your marker (X or O).
In PvC mode, the AI automatically plays its turn.
The first player to align 3 markers horizontally, vertically, or diagonally wins.
If all cells are filled without a winner, the game ends in a draw.
Press Restart to reset the game at any time.
